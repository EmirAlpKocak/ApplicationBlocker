Password

When you launch the Application Blocker for the first time, the tool will ask you to set an User Interface (UI) password. You can set any password you want. This password will be asked when you try to launch Application Blocker. The password is encrypted and stored securely.

Changing Password

You can change your password by clicking "More Options" -> "Change Password". The program will ask for your current password. After entering your current password, you can change your password to anything.
